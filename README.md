## ğŸ’» Getting Started

### Prerequisites

- Node.js >= 15.x
- npm or yarn

### Installation

1. Clone the repo

```bash
git clone git@github.com:syedomer17/Next.js-portfolio.git
cd your-portfolio
````

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. Run the development server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
